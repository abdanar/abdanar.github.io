## Problem 1
If we have the initial value problem

$$
\dot{x}(t) = x(t) + \sum_{i=1}^{n}x(t_{i}),\quad x(t_{j}) = x_{0},
$$

where $x_{0} \in \mathbb{R}$, $t_{i},\ i=1, \ldots, n$, and $t_{j}$ are given, then the solution to the initial value problem is given by

$$
x(t) = x_{0}\left(1+e^{t}+\dfrac{(n+1)e^{t_{j}}}{(n+1)e^{t_{j}} - \sum\limits_{i=1}^{n}e^{t_{i}}}\right).
$$
